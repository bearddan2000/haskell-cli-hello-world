# haskell-cli-hello-world

## Description
A POC for haskell.

## Tech stack
- ghc

## Docker stack
- haskell:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
